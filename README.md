
# Inventory Management System

A Full Stack Inventory Management System built using React, Spring Boot, and MySQL.

## Features

✅ Add Product  
✅ View Product List  
✅ Search Product  
✅ Update Product  
✅ Delete Product  
✅ Dashboard Analytics  
✅ Loading Indicator  
✅ Validation  
✅ Responsive UI

## Installation

### Backend

Clone repository:

```bash
git clone https://github.com/Suchithra09/inventory-management-system
```

Go to backend:

```bash
cd backend
```

Run:

```bash
mvn spring-boot:run
```

Backend:

```text
http://localhost:8080
```

---

### Frontend

Go to frontend:

```bash
cd frontend
```

Install:

```bash
npm install
```

Run:

```bash
npm start
```

Frontend:

```text
http://localhost:3000
```
